Método de transporte Canto Noroeste

Trabalho prático de Pesquisa Operacional - Sistemas de informação - UniBH

  • Desenvolvedor: Davidson Silva - 11424797
  • Professor: Felipe Leandro Andrade

Essa versão do Método Canto Noroeste foi desenvolvida em JavaScript com o auxílio da linguagem de Marcação HTLM5 e os estilos CSS3, visando proporcionar uma melhor experiência para realizar os cálculos necessários e exiir os resultados ao usuário.

Nessa versão calculamos o custo para a distribuíção de estoque para diversas demandas. A princípio foi realizado apenas o cálculo para distribuições equilibradas. Foram criados os métodos createTable(rows, cols), onde o usuário define a quantidade de linhas (rows) de estoque e colunas (cols) de demanda, esses valores são inputados em campos especificados na interface dando assim liberdade e dinâmica ao programa. Após criar a tabela o usuário poderá excluí-la quando achar necessário e poderá criar outra com as dimensões que necessita.

Após criar a tabela o usuário deverá informar os custos de cada combinação e os valores das demandas e estoques disponíveis. Não será possível calcular caso um dos campos de demanda ou estoque estiverem vazios ou forem iguais a 0 (zero), pois se ambos foram declarados logo têm algum valor.

Foi criado também um método que calcula a distribuião do estoque para as demandas cadastradas calcula(rows, cols). O cálculo só é realizado caso a solução seja equilibrada (A normalização não foi implementada nessa versão do programa). Nesse método são realizadas ass equações e comparações necessárias para que a distrribuição seja aplicada a qualquer distribuição equilibrada informada ao programa. Após calcular toda a distribuição o sistema gera a tabela com a combinação encontrada e o custo total de acordo com os custos anteriormente informados pelo usuário.
